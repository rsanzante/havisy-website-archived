HAVISY Guitar & Bass Stickers

Muy pronto tendremos disponibles para descarga y en nuestra tienda nuevas Stickers HAVISY, para guitarra eléctrica, guitarra clásica y bajo eléctrico. Estas stickers estarán disponibles en dos versiones:

-Notas con color HAVISY + intervalos relativos. Ideales para iniciarse si no conoces bien la relación entre las cuerdas.

-Notas con color HAVISY. Si ya conoces la relación entre las cuerdas y quieres pasar al siguiente nivel.

Iremos sacando Stickers para más instrumentos, así como mapas HAVISY para los instrumentos en los que no se puede usar la vista para localizar las notas.
